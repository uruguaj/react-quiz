# Quiz in React

Hi this is my project Quiz in React. 

## Instalation 🛠️
1. Clone the dir
2. Cd in to the dir and type ```npm i``` to install everything you need
3. And now run ```npm run dev``` , your web is running on localhost:5173 (defualt)
